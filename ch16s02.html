Explora més

  • La biblioteca estàndard de Python és força extensa. La major part de les vegades, hi trobarem el que estem buscant. És el que es coneix com la filosofia de "tot inclòs" de Python. Us recomano molt que us passeu per la Documentació estàndard de Python abans de començar a escriure programes de certa grandària.

  • Python.org - és la pàgina oficial del llenguatge de programació Python. Hi trobem les darreres versions del llenguatge i l'intèrpret. També hi ha varies llistes de correu on es discuteix activament sobre diferents aspectes de Python.

  • comp.lang.python és el grup de notícies usenet on es discuteix sobre aquest llenguatge. Hi podem exposar els nostres dubtes i consultes. El trobem a Google Groups i també afegint-nos a la llista de correu què és una imatge d'aquest grup de notícies.

  • Python Cookbook és una col·lecció de receptes i consells per a resoldre diferents problemes amb Python. És de lectura obligatòria per tothom que sigui usuari d'aquest llenguatge.

  • Charming Python ofereix una sèrie d'articles d'en David Mertz, relacionats amb Python

  • Dive Into Python és un llibre molt bo per a programadors experimentats en Python. Si has seguit en profunditat aquest llibre, et recomano que llegeixis 'Dive Into Python'. Cobreix un ampli ventall de temes incloent processament XML, prova unitària i programació funcional.

  • Jython és una implementació de l'intèrpret Python amb Java. Això vol dir que podem escriure programes en Python i fer servir les llibreries de Java! Jython és un programari estable i madur. Si ets programador de Java, et recomano que li facis una ullada.

  • IronPython és una implementació de Python en C# i funciona sobre les plataformes .NET / Mono / DotGNU. Això vol dir que podem escriure programes en Python i usar les llibreries d'aquestes tres plataformes! IronPython és encara en versió pre-alpha i només està recomanat per experimentació. Jim Hugunin, el desenvolupador de IronPython, s'ha incorporat a Microsoft i treballarà en un futur per una versió completa d'aquesta implementació.

  • Lython és una interfície amb Lisp del llenguatge Python. És semblant a Common Lisp i compila directament al codi intermedi de Python de manera que pot interoperar amb el codi general de Python.

  • Hi ha molts i molts més recursos sobre Python. Alguns d'interessants són Daily Python-URL! que ens informa de les darreres notícies sobre el món Python, Vaults of Parnassus, ONLamp.com Python DevCenter, dirtSimple.org, Python Notes i molts i molts més.
